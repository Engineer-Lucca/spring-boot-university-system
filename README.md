# 🎓 spring-boot-university-system

Este repositório contém o desenvolvimento de um sistema de gestão universitária, consolidado como parte das atividades práticas da disciplina de **Engenharia de Software III (ESN3)** na **UNIFAN**. O projeto utiliza a arquitetura robusta do ecossistema Java para simular fluxos acadêmicos e administrativos.

## 🛠️ Tecnologias e Ferramentas

| Categoria | Tecnologia |
| :--- | :--- |
| **Linguagem** | Java 17+ |
| **Framework** | Spring Boot 3.x |
| **Build & Dependency Management** | Maven |
| **Testes de Integração** | JUnit 5 / Spring Boot Test |
| **Versionamento** | Git |
